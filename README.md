# Project setup

1. Install Docker
2. Install PSQL shell
3. Install Git
4. Clone Repo
5. Create .env file with `POSTGRES_USER=your_username_here` and `POSTGRES_PASSWORD=your_password_here`
6. Run `docker compose up` to launch server
7. Connect from the command line with `psql -U your_username_here -d librarydb`, and enter your password when prompted.
